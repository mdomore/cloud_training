# Phase 1: Foundations - Exercises

This directory contains practical exercises for the Foundations phase.

## Structure

- **[01-linux-fundamentals](./01-linux-fundamentals/)** - File system, users, permissions, processes
- **[02-shell-scripting](./02-shell-scripting/)** - Bash/Zsh scripting exercises
- **[03-system-configuration](./03-system-configuration/)** - Network, packages, services, logs
- **[04-security-basics](./04-security-basics/)** - SSH, firewalls, access control
- **[05-networking](./05-networking/)** - Network protocols, DNS, routing
- **[06-development-basics](./06-development-basics/)** - Git, programming, APIs

## How to Use

1. Read the corresponding section in `docs/01-foundations.md`
2. Complete exercises in order (they build on each other)
3. Check your work against the solutions (if provided)
4. Document your learnings and challenges

## Prerequisites

- Access to a Linux system (physical or virtual)
- Basic command-line knowledge
- Text editor (vim, nano, or VS Code)
